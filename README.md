# Catch-Ur-Crayon  

## Explanation

### About Game.
우선 Catch-Ur-Crayon은 넷마블사의 '캐치마인드'를 모작한 게임이다.
게임 방식은 이렇다. 방에 참여한 사람들이 한 사람씩 돌아가며 출제자가 된다. 출제자는 주어진 단어를 캔버스에 그림으로 그려서 묘사하고, 나머지 사람들은 그 단어를 맞히면 되는 것.
